## Run Locally

Clone the project

```bash
  git clone https://github.com/NutriaSoft/nutriasoft.dev
```

Go to the project directory

```bash
  cd nutriasoft.dev
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
